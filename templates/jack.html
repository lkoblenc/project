Projects

Street Mapping of New York State

Using Java, I created a graph data structure to represent the intersections and roads of New York State. After constructing the graph, the coordinates of the intersections were used to create a map that was displayed to the user. Dijkstra's Algorithm was implemented to find the shortest route from a given intersection to all other intersections, and Kruskal's Algorithm was implemented to find the minimum weight spanning tree given an intersection.

Connect Four Application

I used Java, JavaFX, and Scene Builder to design the classic game of connect four. Two players take turns placing tokens into columns. The application also allows the users to change the dimensions of the playing board, as well as the minimum amount of tokens in a row it takes to win a game.

Website with custom content management system.

The website was a collection of student's about-me pages utilizing MySQL, HTML, PHP, CSS. The website had a custom content management system (CMS) and login interface, and the website allowed users with a valid username and password to create, edit, and delete student pages on the website.
